# Changelog - OfflineSync Pro

## [1.0.0] - 2025

### Adicionado
- Sistema completo de sincronização offline-first
- Backend Node.js com Express e SQLite
- Flutter app com SQLite local
- Sync Engine com fila de operações
- Detecção e resolução de conflitos (Last Write Wins)
- Autenticação JWT
- CRUD de exemplo funcional
- Documentação completa
- Guia de produção

### Características
- Persistência offline real
- Sincronização bidirecional
- Controle de versão
- Retry automático
- Soft delete
- Status de sincronização visual

### Limitações Conhecidas
- Estratégia de conflito: Last Write Wins
- Uma entidade de exemplo (fácil de replicar)
- SQLite no servidor (pode migrar para PostgreSQL)

**Versão inicial - Pronta para uso e adaptação**
