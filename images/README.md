# Imagens do SyncFlow Engine

Esta pasta contém as imagens do SyncFlow Engine para a landing page.

## Como adicionar as imagens:

1. **Copie as imagens da pasta Downloads:**
   - `ChatGPT Image 14 de jan. de 2026, 17_55_43.png`
   - `ChatGPT Image 19 de jan. de 2026, 19_11_03.png`

2. **Renomeie as imagens:**
   - `syncflow-engine-1.png` (primeira imagem - visão geral)
   - `syncflow-engine-2.png` (segunda imagem - fluxo de sincronização)

3. **Cole nesta pasta:** `images/`

## Estrutura esperada:

```
images/
├── syncflow-engine-1.png  (Arquitetura e Funcionalidades)
├── syncflow-engine-2.png  (Fluxo de Sincronização)
└── README.md              (este arquivo)
```

## Nota:

Se as imagens não forem encontradas, a landing page ainda funcionará normalmente, 
mas as imagens não serão exibidas. O código HTML já está preparado para isso.
